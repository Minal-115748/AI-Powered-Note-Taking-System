# LumenNote Studio

This is the proper React + Vite version of your note-taking app for VS Code.

## Run in VS Code

1. Open the folder `D:\niraj\vite-project`
2. Open the terminal in VS Code
3. Run `npm install`
4. Run `npm run dev`
5. Open the local URL shown in the terminal

## Features

- Vertical navbar
- Notes with search, tags, pin, and delete
- To-do planner
- Voice notes
- Record upload section
- Calendar with color events
- Resume builder
- Settings, login, and logout

## Data

- Seed data lives in `src/data/seed.json`
- Runtime changes are saved in browser `localStorage`
