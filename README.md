# 🧠 AI Powered Note Taking System

An all-in-one productivity web application built during Technovate’26 Hackathon.  
This project helps users create, manage, and enhance notes using AI-powered features.

---

## 🚀 Features

- 🧠 AI-based Note Summarization  
- 🎤 Voice Assistant for Note Creation  
- 📝 Markdown Support  
- 📊 Flowchart Creator  
- 📄 Resume Builder  
- 📅 Calendar Integration
