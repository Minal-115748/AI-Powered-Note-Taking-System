# LumenNote Studio

A React-based note-taking web app with a vertical navbar and a richer visual style inspired by your sketch.

## Features

- Notes with categories, tags, pinning, search, and delete
- To-do planner with priority, due date, complete, and delete
- Voice notes using browser speech recognition
- Record center for image, video, and audio attachments
- Calendar with color-coded events
- Resume builder with live preview and print/export
- Settings, profile, privacy, contact, login, and logout views

## Data

- `data/seed.json` is the starter JSON database
- The app saves updates in browser `localStorage` under `lumen-note-studio`

## Run

Open `index.html` in a browser with internet access so the React and Babel CDN files can load.
